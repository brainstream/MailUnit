MailUnit

Warning: it is early unstable development version!

MailUnit allows to test SMTP clients and request sent messages using the mail query protocol (MQP).

Download

Now you can only download MailUnit's source code from the github

Download source code

or clone the git repository using following command

git clone https://github.com/brainstream/MailUnit.git

Build

MailUnit has following dependencies

  • Boost.Filesystem
  • Boost.Date_Time
  • Boost.Program_Options
  • Boost.Regex
  • Boost.Iostreams

Also a C++11 supported compiler is required.

MailUnit uses CMake building system. You should enter following command to build binary files to a suitable place.

cd MailUnit
cmake .
make

It will create two files: the executeble file named mailunit and library with the same name.

Install

SMTP server

Only the main SMTP commands are supported now: EHLO, MAIL FROM, RCPT TO and QUIT.

MQP server

MQP (Mail Query Protocol) allows to query received by SMTP server message using simple query language.
